# ImageClassifier
A simple image classification neural network

## Prerequisites
 Install Tensorflow and Kers

- https://www.tensorflow.org/
- https://keras.io/

```
pip install keras
pip install tensorflow
pip install pillow
```

### Create test and training set
copy your images to the following folder structure

```
/dataset
    /test_set
        /category1
        /category2
    /training_set
        /category1
        /category2
```
